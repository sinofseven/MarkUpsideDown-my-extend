# Tags & Organization

Tags let you categorize and filter files with colored labels. You can assign multiple tags to any file and use them to quickly find related content.

## Creating a Tag

1. Right-click a file in the sidebar
2. Select the tag option to open the **tag popover**
3. Type a tag name (up to 24 characters) in the "New tag…" input
4. Choose a color from the 8 available swatches (Red, Orange, Yellow, Green, Blue, Purple, Pink, Gray)
5. Click **Add**

## Assigning Tags

In the tag popover, each existing tag has a checkbox:

- **Checked** — the file has this tag
- **Unchecked** — the file does not have this tag
- **Indeterminate** — when multiple files are selected and only some have the tag

Click the checkbox to toggle the tag on or off for the selected file(s).

## Multi-File Tagging

Select multiple files (Cmd+Click or Shift+Click), then right-click and open the tag popover. Tags are applied to all selected files at once.

## Tag Chip Bar

The sidebar shows a row of colored tag chips below the search input. These act as quick-filter toggles:

- Click a tag chip to filter the file tree to only show files with that tag
- The active filter is shown as a "tag: TagName" badge with a ✕ button
- Click ✕ to clear the filter

## Visual Indicators

Files with tags show colored dots next to their names in the sidebar, making it easy to see at a glance which files are tagged.

## Sorting by Tag

Use the sort dropdown at the top of the Files panel and select **Tag** to group files by their tag assignments.

## Deleting a Tag

In the tag popover, click the **✕** button next to a tag name to delete it. This removes the tag from all files.
